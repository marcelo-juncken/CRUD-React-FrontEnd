# CRUD de Usuários

## Instalação

No diretório do projeto execute o comando:

### `npm install`

ou

### `yarn install`

## Como Utilizar

Após a instalação, ainda na pasta raiz, rode o comando:

### `npm start`

ou

### `yarn start`
